PUF Hardware Engineering

A two-way multimedia authentication system built on FPGA using physically unclonable functions (PUFs), enabling secure identity verification through unique, tamper-proof hardware signatures.

OEG Human AI Interaction Robotics

I developed online explanation generation (OEG) for human-robot teaming, reducing cognitive load by incrementally delivering model-reconciling justifications during task execution. Evaluated across domains with strong empirical results

PEG Human AI Interaction Robotics

I introduced progressive explanation generation (PEG) for human-robot teaming, modeling explanation order preferences via inverse reinforcement learning to reduce cognitive load and improve interpretability during task execution.

NOVA Robotics Human AI Interaction
